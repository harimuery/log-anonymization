  - `log-anonymization-core/src/main/java/com/example/anonymization/core/infrastructure/version/RuleVersionCompatHandler.java`：规则版本兼容处理器
- **实现内容**：
  - **Version 值对象**：SemVer 2.0.0（MAJOR.MINOR.PATCH），支持 parse("1.0.0")、isCompatibleWith()、isNewerThan()、isOlderThan()
  - **VersionCompatPolicy 接口**：isCompatible(v1, v2)、isDeprecated(version)、isSupported(version)、getCurrentVersion()

  - **YamlRuleParser 集成**：解析 YAML 头部 version 字段 → Version 对象 → versionHandler.handle() 兼容性检查
  - **AutoConfiguration 集成**：versionCompatPolicy / ruleVersionCompatHandler / grayReleaseRouter 三个 Bean
  - **GrayReleaseConfig**：灰度比例 percent（0-100）、oldVersion、newVersion 配置
